# F7 — Meeting Summary

## Feature Description

Generate a structured AI summary of each meeting containing Key Discussion, Decisions, Risks, and Action Items. Uses the local LLM model (Qwen2.5-1.5B) via `llm.py`. Falls back to a rule-based summary if the model is unavailable or inference fails. Summaries are persisted in the `analyses` table and loaded from storage on subsequent visits.
