# F11 — Cross-Meeting Knowledge Graph

## Feature Description

Visualize relationships between people and action items across meetings. Uses NetworkX to build a directed graph from existing action items data, with edges from owners to their action items. Skips "Unassigned" owners. Rendered with matplotlib.

---

| Field | Value |
|-------|-------|
| Input | Owner="Alice", Item="Create database index" |
| Expected Result | Graph contains one edge: Alice → Create database index |
