# F1 — Upload Meeting Notes

## Feature Description

Users can submit meeting notes via four methods:
1. Typing notes directly into a text area.
2. Uploading a `.txt` file.
3. Uploading a `.md` (Markdown) file.
4. Uploading a `.pdf` file.

Extracted text is stored locally in an SQLite database (`meetings` table).
