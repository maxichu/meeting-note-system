# F10 — Save All Analysis

## Feature Description

Persist all extracted analysis results (speakers, action items, owners, deadlines) as a single JSON blob in the `analyses` table. Displays from stored analysis by default in Meeting History, falling back to individual tables for legacy data.

| Input | Meeting with `Alice` and `Bob` as speakers, action items with owner and deadline. |
| Expected Result | Analysis saved successfully. History view reads from `analyses` table. |
