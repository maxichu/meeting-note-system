# F6 — Deadline Detection

## Feature Description

Automatically detect deadlines associated with action items. Uses a two-priority rule cascade: (1) explicit date patterns (`2026-06-15`, `2026/06/15`), (2) relative date keywords (`tomorrow`, `next week`, `next month`, day names). Deadline expressions are removed from the action text and stored separately.

---

## Test Case 1 — Relative Keyword

| Field | Value |
|-------|-------|
| Input | `Action item:\nCreate database index by Friday.` |
| Expected Result | Deadline = "Friday", cleaned text = "Create database index." |
